# د. بشار للسياحة - Dr. Bashar for Tourism

موقع ويب احترافي لشركة سياحة متخصصة في روسيا.

## وصف الموقع

موقع ويب عصري وجذاب لشركة "د. بشار للسياحة" التي تقدم خدمات سياحية للسياح العرب في روسيا. يتضمن الموقع معلومات عن الشركة، خدماتها، فعالياتها، وطرق التواصل معها.

## المميزات

- تصميم عصري وجذاب
- واجهة مستخدم سهلة الاستخدام
- متوافق مع جميع الأجهزة (تصميم متجاوب)
- معرض صور وفيديوهات تفاعلي
- نموذج تواصل مع العملاء
- عرض للفعاليات والخدمات

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome (للأيقونات)
- Google Fonts (للخطوط)
